Wordpress Responsive News Scroller 

== Description ==

A Simple Wordpress plugin which is used to scroll news in responsive mode. To display any where in post or pages use this short code [wpr_scroller]

For use in template `<?php  wpr_func($atts) ; ?>` 

**Features**

* Responsive
* Work with all browsers
* Set Number of news to scroll
* Add unlimited news category
* Add unlimited news

== Installation ==


1. Upload `wpr-scroller.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Shortcode to display news scroller in  post are pages [wpr_scroller]
1. Place `<?php  wpr_func($atts) ; ?>` in your templates
